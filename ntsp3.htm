Windows NT Service Pack 3

This Service Pack compact disc contains the standard Windows NT(R) Service Pack and updates to Windows NT features and products that ship with Windows NT.

Internet Information Server (IIS) 3.0 provides an upgrade to the Windows NT Server 4.0 / IIS 2.0 feature set. IIS 3.0 extends the functionality of IIS 2.0 with the following new features. You may install one or all of these new features:

  • Active Server Pages 1.0b
  • Microsoft Index Server 1.1
  • Microsoft NetShow 1.0
  • Microsoft FrontPage(TM) 97 Server Extensions
  • Crystal Reports

The updated products that ship with Windows NT are:

  • FrontPage 97
  • Microsoft Internet Explorer 3.02

Some of the setup programs on this compact disc will require you to reboot your computer. To see this page after rebooting, you may need to eject the compact disc and then replace it in the drive.

To properly install software using the links on this page, you must click "Open It" on the browser dialog box.

The links to the various setup programs on this page are intended for use with Microsoft Windows NT 4.0 and Internet Explorer. Please install directly from the product and platform directories on the compact disc if you are installing on a different operating system/browser configuration.


Windows NT Service Pack

This release of Microsoft Windows NT 4.0 Service Pack 3 is easy to apply from within Windows NT, and it changes only those files that were originally set up on the Windows NT Workstation or Windows NT Server operating systems. Service Pack releases are cumulative; they contain all previous fixes, as well as any new fixes made to the system.

Service Pack 3 upgrades the Internet Information Server (IIS) core to version 3.0 as well as providing a few new features. The Service Pack provides the hooks needed for the new features in IIS 3.0. Your IIS version number will change to 3.0 after installing the Service Pack. To install the optional new features of Internet Information Server 3.0, see the IIS 3.0 section below.

Service Pack 3 also provides the necessary functionality for the Microsoft NetMeeting 2.0 Windows NT Client. To download Microsoft NetMeeting 2.0, see http://www.microsoft.com/netmeeting/.

If you are using the 128 bit North American version of Windows NT, read this important information before installing the Service Pack.

It is very important that you read the release notes before installing this Service Pack.

Internet Information Server 3.0

Internet Information Server 3.0 makes it easier to build server-based Web applications and to create rich and dynamic content. It also includes support for multimedia streaming, enhanced context indexing, and improved site management for corporate intranets and Internet sites.

You may install any combination of the features listed below. You must have IIS 2.0 installed in order to upgrade to IIS 3.0. Before you install any of the features below, you must first install:

  • IIS 2.0 from Windows NT 4.0 compact disc
  • Windows NT Service Pack 3.0 as described above

For more information on any of the features listed below, see our website at http://www.microsoft.com/iis/.

Active Server Pages 1.0b
Formerly code-named "Denali," Active Server Pages is an open, compile-free application framework that allows organizations to combine HTML, scripts, and components to rapidly build powerful Web-based applications. Active Server Pages includes native support for Microsoft Visual Basic(R) Scripting Edition (VBScript) and JScript. With Active Server Pages, many more people can write server-side Web applications for the first time.

Release notes

Install Active Server Pages

Microsoft Index Server 1.1
Index Server is a built-in search engine that provides full-text indexing, searching, and hit highlighting of many types of information, in a variety of formats including HTML, Microsoft Office, or text documents. Index Server provides a way to quickly locate specific information on intranet or Internet sites.

Install Index Server 1.1

 Microsoft NetShow 1.0
Microsoft NetShow is the software platform for one-to-many multimedia communications and information distribution over the Internet and corporate intranets. NetShow technologies enhance users' networked multimedia experience by providing live and on-demand audio and video content, while reducing impact on network bandwidth. Please review the release notes before installing.

Several different packages are available:

NetShow Live

•Server, Client, Documentation, and SDK

NetShow On-Demand

•Content Creation Tools (available in x86 only)
•Server
•Player (available in x86 only)
•SDK and Documentation in HTML Format
•Printable SDK and Documentation in RTF Format
•Demos Suite (available in x86 only)

FrontPage 97 Server Extensions
The FrontPage 97 Server Extensions allow one-button publishing and graphical site-management tools to keep information organized. Users may now publish to the site from the desktop without changing links, graphically locate broken links, and create links between HTML and Microsoft Office files. FrontPage 97 Server Extensions enable organizations to develop their corporate intranet and Internet sites quickly and manage them more effectively.

Install FrontPage 97 Server Extensions

Crystal Reports
Crystal Reports is a client/server report writer that you can use to create presentation-quality reports and integrate them into database applications. Crystal Reports for Internet Information Server includes features to create reports from your Web Server log files and includes pre-formatted Web log reports. HTML publishing capabilities instantly turn reports into presentation-quality, information-rich documents for the Internet and intranet.

Before installing Crystal Reports, you must install Active Server Pages as specified above.
